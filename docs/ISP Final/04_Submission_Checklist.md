# ISP Final Submission Checklist

Use this checklist after preparing final documents.

## A. Final Report Quality Check

- [ ] Used IE2091 Final Report Template.doc structure
- [ ] Added IE2091 Final Report Cover Page.doc details
- [ ] Included full team names and student IDs
- [ ] Included problem statement and business context
- [ ] Included architecture and technology stack
- [ ] Included all six epics with implemented scope
- [ ] Included testing evidence (screenshots/API results)
- [ ] Included challenges, lessons learned, conclusion, future work
- [ ] Grammar and formatting reviewed
- [ ] Exported to required format (DOCX/PDF as requested)

## B. Final Presentation Quality Check

- [ ] Followed IE2091 Final Presentation Guidelines.pdf
- [ ] Clear slide narrative from problem -> solution -> impact
- [ ] Slides contain readable visuals and no clutter
- [ ] Demo flow rehearsed and time-boxed
- [ ] Team contribution slide included
- [ ] Risks/challenges and mitigation slide included
- [ ] Final Q&A slide included

## C. Marketing Video Quality Check

- [ ] Followed IE2091 Marketing Video Guidelines.pdf
- [ ] Video duration meets required limit
- [ ] Includes product value proposition and key features
- [ ] Includes customer + staff workflow highlights
- [ ] Audio is clear; subtitles included
- [ ] No private data or credentials shown
- [ ] Exported to required format/resolution

## D. Common Mistakes Prevention

Cross-check against:
- IE2091 Common issues to address in the projects.pdf

- [ ] No missing references between sections
- [ ] No mismatch between slides, report, and demo claims
- [ ] No unfinished placeholders (e.g., [Fill])
- [ ] No contradictory numbers/dates/team details
- [ ] No broken screenshots or unreadable text

## E. Final Packaging

- [ ] Final report file renamed as instructed
- [ ] Presentation file renamed as instructed
- [ ] Marketing video file renamed as instructed
- [ ] All files placed in one submission folder
- [ ] Final quick review done by at least 2 team members

## F. Suggested Final Submission Folder Layout

- ISP_G05_Final_Report.docx
- ISP_G05_Final_Presentation.pptx
- ISP_G05_Marketing_Video.mp4
- ISP_G05_Readme.txt (optional)
