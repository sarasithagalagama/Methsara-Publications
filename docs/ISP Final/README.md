## 4. How to use this folder now

1. Build final report
- Open IE2091 Final Report Template.doc
- Copy the full content from 01_Final_Report_Draft.md section by section
- Add the official first page from IE2091 Final Report Cover Page.doc
- Export to the required final format

2. Build final presentation
- Create slides in PowerPoint using 02_Final_Presentation_Outline.md
- Use one section per slide and keep the same order
- Include screenshots/evidence as specified in the file

3. Produce marketing video
- Follow 03_Marketing_Video_Script.md scene by scene
- Record voiceover using the included final script
- Apply the included production and compliance checklist

4. Run final QA
- Complete every item in 04_Submission_Checklist.md before submitting
